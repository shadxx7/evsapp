Instructions

  • Enter ranges for the six parameters in the corresponding text fields provided. Range input should be given in the format 'lower_bound-upper_bound' Make sure there is no trailing whitespace and no space between the bound values and the hyphen delimiter. To pass a decreasing range as the parameter input, set the value of the lower_bound to be greater than that of the upper_bound. To pass a constant input for a particular parameter, set both lower_bound and upper_bound to the same value. Make sure that no parameter input is left blank.
  • The number of days specify the timeframe over which the range inputs are varied. Broadly speaking, the parameter input ranges are varied uniformly across the duration of the timeframe to simulate it's proper effect on the Hydrological Cycle Process.
  • Click on the Field Name of each parameter to get more information about that particular component's role in the Hydrological Cycle.
  • Press Calculate to render a dynamic, scalable and draggable graph plotting the value of precipitation against the varying ranges of the other components in the Hydrological Budget Equation.
  • Use Ctrl+R to Reset the current application buffer data and make it ready to accept a new set of input parameter ranges and return output for a new graph simulation.
